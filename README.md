# E-Commerce Sales Dashboard (Power BI)

## Project Overview
This project analyzes e-commerce sales data using Power BI to identify revenue trends, customer behavior, payment preferences, and delivery performance.

## Key Insights
• Total Revenue: 16M+
• Total Orders: 99K
• Average Order Value: 160
• Credit card is the dominant payment method
• Delivery time improved between 2016 and 2018

## Dashboard Features
• Revenue trend analysis
• Revenue by payment method
• Customer segmentation
• Order status distribution
• Delivery performance tracking

## Tools Used
• Power BI
• SQL
• Data Modeling
• DAX Measures
